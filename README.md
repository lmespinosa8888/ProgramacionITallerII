# Proyecto Python - Sistema de Biblioteca Escolar

Este proyecto demuestra el uso de:

- Instancia de objetos
- Creación de métodos
- Uso del método constructor

## Clases principales

- `Libro`
- `Usuario`
- `Biblioteca`

## Cómo ejecutar

1. Abre una terminal en la carpeta del proyecto.
2. Ejecuta:

```bash
python main.py
```

## Qué se muestra

- Se crean objetos `Libro` y `Usuario`.
- Se usan constructores para inicializar atributos.
- Se ejecutan métodos para prestar y devolver libros.
- Se imprime el estado de la biblioteca antes y después de las operaciones.
